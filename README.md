# SOC Investigation Portfolio

## Overview

Welcome to my Security Operations Center (SOC) Investigation Portfolio.

This repository contains documented cybersecurity investigations based on simulated attack scenarios. Each case study demonstrates my ability to analyze alerts, investigate suspicious activity, and reconstruct attack chains using security logs and monitoring tools.

The goal of this portfolio is to showcase practical SOC analyst skills, including threat detection, incident analysis, and incident reporting.

# Investigation Case Studies

1. Phishing Attack Investigation

### Description:
Analysis of a phishing attack that resulted in PowerShell execution, reverse shell connection, and suspicious DNS activity.

### Key Findings:

- Malicious phishing email triggered initial compromise
- PowerShell used to download payload
- Reverse shell established with external server
- Suspicious DNS queries indicating command-and-control communication

2.  Malware Analysis

### Description:
Investigation of suspicious files to understand malware behavior and identify indicators of compromise.

### Focus Areas:

- Static analysis of malicious files
- Identifying suspicious strings and file characteristics
- Observing malware behavior
- Extracting indicators such as hashes, domains, and IP addresses

3. Network Traffic Investigation

### Description:
Analysis of network traffic logs to identify suspicious communication between compromised systems and external servers.

### Focus Areas:

- Identifying unusual outbound connections
- Detecting command-and-control traffic
- Investigating DNS requests
- Reviewing packet capture (PCAP) data

4.  SIEM Alert Triage

### Description:
Investigation and prioritization of alerts generated by Security Information and Event Management (SIEM) systems.

### Focus Areas:

- Analyzing security alerts
- Determining false positives vs real threats
- Correlating multiple events
- Escalating confirmed incidents
